# Student Management REST API

This project is a backend REST API built using Java and Spring Boot to manage student records.

## Features
- RESTful API design
- CRUD operations for students
- Clean layered architecture (Controller, Service, Model)
- HTTP methods: GET, POST, PUT, DELETE

## Tech Stack
- Java
- Spring Boot
- Maven
- REST API
- MySQL (conceptual)

## API Endpoints
- GET /students
- POST /students
- PUT /students/{id}
- DELETE /students/{id}
